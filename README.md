# Amor Medical EMR v3

**Complete Electronic Medical Records Platform**

Built by Ashley 💜 for Dr. Roger Gonzalez and Amor Medical

## Features

### Core EMR
- ✅ Patient Management (full demographics, insurance, pharmacy)
- ✅ Provider Management (NPI, DEA, license, specialty)
- ✅ Practice/Office Management
- ✅ Encounters with SOAP Notes
- ✅ Voice Dictation (Web Speech API)
- ✅ Vitals Tracking
- ✅ Prescriptions (50+ medication database)
- ✅ Lab Orders (Quest, LabCorp integration prep)

### Clinical Tools
- ✅ 20+ Medical Calculators (A1C, eGFR, BMI, HOMA-IR, ASCVD, CHA2DS2-VASc, etc.)
- ✅ ICD-10 Code Search (60+ common codes)
- ✅ CPT Code Lookup with RVU values
- ✅ Auto PDF Generation for encounters
- ✅ Provider Credentialing Checklist

### Executive Features
- ✅ Executive Dashboard with metrics
- ✅ Revenue estimation (CPT-based)
- ✅ Provider productivity tracking
- ✅ Encounter type breakdown

### Technical
- 📱 PWA - Installable on any device
- 💾 Local Storage - Works offline
- 🖨️ Print-ready encounter PDFs
- 🎙️ Voice dictation for SOAP notes

## Quick Start

1. Open `index.html` in browser
2. Click "Install App" when prompted (or add to home screen)
3. Start adding patients and encounters!

## Data

All data is stored in browser localStorage. To backup:
- Settings → Export JSON

To reset demo data:
- Settings → Reset Demo Data
